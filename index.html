Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

  • Web 2.0: Permite a los usuarios generar contenido y participar en plataformas interactivas como blogs y redes sociales.
  • Web 3.0: Es una web más inteligente que usa tecnologías como la IA y el blockchain para mejorar la experiencia.
  • La W3C y los estándares: Organización que establece reglas para la correcta visualización y accesibilidad de las páginas web.

II. Lenguajes de etiquetas

  • SGML: Lenguaje utilizado para la creación de otros lenguajes de marcas como HTML.
  • XML: Utilizado para almacenar y transportar datos de forma estructurada.
  • HTML: Lenguaje principal para estructurar una página web.
  • XHTML: Versión más estricta de HTML que sigue las reglas de XML.
  • HTML5: Última versión de HTML con soporte para multimedia y gráficos sin complementos.

III. Otros lenguajes relacionados al desarrollo web

  • CSS/CSS3: Lenguaje que se usa para darle estilo a las páginas web.
  • JavaScript: Lenguaje de programación que añade interactividad a la web.
  • Ajax: Técnica que permite actualizar partes de la página sin recargarla por completo.
  • Tecnologías del lado del servidor: Lenguajes como PHP, Python, Ruby que procesan el contenido en el servidor.
  • Navegadores: Programas que permiten navegar en internet (Chrome, Firefox, Safari).
  • Motores renderizados: Motores que los navegadores utilizan para interpretar el HTML y CSS.

IV. Herramientas de desarrollo

  • Software WYSIWYG: Herramientas como Adobe Dreamweaver permiten diseñar sin escribir código directamente.
  • Otras alternativas:
    • Figma
    • InVision
    • Zeplin
    • GitHub
  • Inspección de código desde el navegador: Herramienta para revisar y editar el código fuente en tiempo real.

Tema 2: El Diseño De Una Página Web

I. Tipos de páginas Web

Existen diferentes tipos de páginas web como estáticas, dinámicas, blogs, portales de noticias y tiendas en línea.

II. Puntos básicos en la elaboración de una página Web

En la creación de una página web es importante tener en cuenta el diseño visual, la accesibilidad, la experiencia de usuario (UX) y que sea adaptable a dispositivos móviles (diseño responsive).
